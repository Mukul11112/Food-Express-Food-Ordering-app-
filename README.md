 FoodExpress – Food Delivery Application
A responsive and feature-rich Food Delivery Web Application built using HTML, CSS, JavaScript, and Bootstrap. This project simulates a complete food ordering system including user authentication, restaurant browsing, cart management, order placement, tracking, and admin controls.
 Features:
 #User Features:
1.User Registration & Login system
2.Browse restaurants and menus
3.Search restaurants
4.Add/remove items to cart
5.View cart total dynamically
6.Place orders with delivery details
7.Order tracking system
# Cart & Checkout
1.Real-time cart updates
2.Multiple payment methods (COD, UPI, Card)
3.Address input for delivery
4.Order confirmation messages
# Order Tracking
1.Track order status dynamically
2.View all placed orders
# Admin Panel
1.Add new restaurants
2.Add menu items
3.View all restaurants and menu items

# Tech Stack
Frontend: HTML5, CSS3, JavaScript
Framework: Bootstrap 5
Styling: Custom CSS + Bootstrap
Logic Handling: Vanilla JavaScript

# How It Works
1.Data is managed using JavaScript (in-memory storage / arrays)
2.DOM manipulation is used for:
  a.Rendering restaurants & menu
  a.Updating cart
3.Handling user login/register
4.Bootstrap ensures responsiveness and UI consistency
